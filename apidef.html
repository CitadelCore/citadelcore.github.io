Coordinator

The RackView Coordinator Northbound API allows you to perform operations related to the management and control of cabinets, cabinet supervisors, and the coordinator itself.

All configuration is done through a namespace system called SRN (Supervisor Resource Name).

Srn Repository  ¶

Get Value
GET/config/{srn}

This endpoint retrieves the value at the specified SRN from the Coordinator.

In the case that the SRN is not fully qualified, the Coordinator will attempt to return a JSON object containing the full scope of the SRN. For certain providers (i.e. vault) this is not supported. Please see the documentation of the specific provider and/or route for more info.

Passing “any” as the provider here or specifying multiple providers will attempt to use all available or the providers you have passed to retrieve the value if it is not found at one. Partially qualified SRNs are not supported.

Example URI

GET /config/urn:srn:v0:global:test

Create Supervisor
POST/supervisors/{id}

This action allows you to pre-stage a Supervisor in the database for later discovery and auto-enrollment. By pre-staging a device you eliminate the act of manually approving it.

Alternatively, you may provide a hostname of a Supervisor in Initialisation Mode to directly enroll it.
